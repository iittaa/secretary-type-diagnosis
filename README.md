# あなたの秘書タイプ診断

オンライン秘書スクールの集客用Web診断アプリ。Instagramを起点としたプレゼントキャンペーン（特典配布）で使用します。

**本番URL**: https://secretary-type-diagnosis.vercel.app

## 概要

15の質問に答えるだけで、あなたにぴったりの秘書タイプがわかる診断コンテンツです。

### 診断結果（4タイプ）

| タイプ | キャッチコピー |
|--------|---------------|
| 💗 ホスピタリティ秘書 | 心で寄り添う癒しの右腕 |
| 👑 ディレクター秘書 | 全体を回す頼れる司令塔 |
| 🎨 クリエイティブ秘書 | センスで魅せるSNSの達人 |
| 🔮 リサーチ秘書 | 正確さで支える情報参謀 |

### 画面フロー

```
スタート → 質問（15問）→ AI分析中 → 診断結果 → アンケート → 完了
                                               └ スキップ → 完了
```

## 技術構成

- HTML + CSS + JavaScript（単一ファイル: `index.html`）
- 外部依存: Google Fonts（Zen Maru Gothic）のみ
- サーバー不要の静的サイト
- ホスティング: Vercel

## デザイン

- スマホファースト（Instagramユーザー向け）
- ベージュ基調のかわいい系デザイン
- メインカラー: `#D4A574` / 背景: `#FDF8F3`
- フォント: Zen Maru Gothic（丸ゴシック）

## ローカルで動かす

```bash
python3 -m http.server 8080
```

ブラウザで http://localhost:8080 を開く。

## デプロイ

```bash
vercel --prod
```

## アンケート機能

診断結果の後にアンケートフォームを表示します（スキップ可能）。

| 項目 | 必須 |
|------|------|
| お名前（ニックネーム可） | ○ |
| Instagram ID | - |
| オンライン秘書への興味度 | ○ |
| ご意見・ご感想 | - |

### Google スプレッドシート連携（任意）

回答をスプレッドシートに保存したい場合は `google-apps-script.js` を参照してください。

1. Google スプレッドシートを作成し、ヘッダー行を入力
2. Apps Script にコードを貼り付け
3. ウェブアプリとしてデプロイ
4. 取得したURLを `index.html` の `GOOGLE_SCRIPT_URL` に設定
